Tour Booking Website

Full-Stack Web App

A complete travel booking platform where users can explore destinations, filter packages, and book tours securely. The application supports both client and admin sides with smooth navigation and role-based interfaces.

Backend & Database

Built with Node.js and Express, connected to MongoDB for storing users, bookings, and destinations. RESTful APIs were used for booking, managing tours, and user operations.

Mail System

Used Nodemailer to send booking confirmations, cancellation receipts, and contact messages. Ensures smooth communication with users and admin through system-generated emails.

JWT Authentication

Implemented secure login/signup with JWT tokens. Differentiated admin vs. user access. Booking, canceling, or editing tours requires valid authentication.

Responsive Frontend

Developed with HTML, CSS, and JavaScript. Designed responsive layouts for all devices. Included filters, form validations, and animated UI sections for smooth user experience.
