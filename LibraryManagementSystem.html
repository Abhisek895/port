Library Management System

Java Swing UI

Designed a user-friendly desktop interface using Java Swing. Features intuitive navigation, form validations, and visual feedback for issuing, returning, and managing books.

JDBC + MySQL Integration

Connected seamlessly with a MySQL database using JDBC. Managed relational tables for Students, Books, Issues, Returns, and History Logs with robust query performance.

Systematic Mail Notifications

Integrated JavaMail API to send system-generated emails such as issue confirmations, return receipts, overdue alerts, and fine reminders to registered student email addresses.

Return System & Fine Calculation

Implemented a smart return module that checks due dates and automatically calculates fine based on days overdue. Supports bulk returns and updates book availability in real-time.

Advanced Search & Logs

Built search filters to find books by name, author, or ID. Tracks every transaction in a logs table for admin review, with options to export reports and data for audits.

Authentication & Access Control

Secured the system with login-based role access (Admin vs Staff). All sensitive actions like delete/edit require admin authentication to prevent data misuse.
