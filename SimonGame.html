Project: Simon Game

Memory-based Simon Game

This is a classic memory game inspired by Simon Says, where users must follow an increasingly complex sequence of lights and sounds. It is built using HTML, CSS, and JavaScript, and is an excellent showcase of event-driven programming, game logic, and user interaction through DOM manipulation.

Project Highlights

• Fully responsive and interactive design
• Pattern logic with increasing difficulty
• Sound and color animations
• Game restart on failure
• Clean code with modular functions
